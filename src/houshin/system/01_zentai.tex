2021年度も同様，会内方針と局内方針の二つを立てる．

\subsubsection*{会内方針}
秋学期は対面活動が再開される可能性があるため，対面活動に対応できるように準備する．

\subsubsection*{局内方針}
以下二つを局内方針とする．
\begin{itemize}
  \item サーバ管理の徹底
  \item クライアントPCのメンテナンス
\end{itemize}
サーバ管理の徹底は会全体にサーバリソースを貸し出せるように整備することや，
NASの外部接続を開始することで達成を目指す．
クライアントPCのメンテナンスは対面活動が再開されたら定期的に行う．
